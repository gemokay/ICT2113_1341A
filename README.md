# ICT2113_1341A

This repository contains project material and a small requirements sub-repo used for the ICT2113 course.

## Folder structure

- **SHMS-reqrs-repo/**: Requirements repository used by the project
	- **BCIC/**: A subfolder (contains its own README)
	- **Models/**: Contains model documents such as the data dictionary ([data_dictionary_v0.1.md](SHMS-reqrs-repo/Models/data_dictionary_v0.1.md))
	- **Requirements/**: Requirement documents
		- **Functional Reqrs/**: Functional requirements (e.g. [FRs_v0.1.md](SHMS-reqrs-repo/Requirements/Functional Reqrs/FRs_v0.1.md))
		- **Non Functional Reqrs/**: Non-functional requirements (e.g. [NFRs_v0.1.md](SHMS-reqrs-repo/Requirements/Non Functional Reqrs/NFRs_v0.1.md))
